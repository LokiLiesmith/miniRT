MOUSE:
LMB: Select object
RMB - hold and drag: Rotate camera
MMB - hold and drag: Pan camera
(on letting go it renders with more samples so more definiton)

Scroll: Zoom
(Move camera in Z)

KEYBOARD:
when selected: 
G - move mode
X - Scale mode
R - Rotate mode

G - MODE: (move)
WASD - moves the selected object
ESC - deselect

S - MODE: (scale)
R - scale radius
H - scale height
W - increase selected parameter
S - decrease selected parameter 

R - MODE (rotate)
WS - rotate around current X-axis
AD - rotate around current Y-axis
QE - rotate around current Z-axis

N: Toggle multi-threading and single-threaded render (gets really slow)

M: Magic
